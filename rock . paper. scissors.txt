1.	معرفی کوتاه
این پروژه یه بازی ساده‌ی سنگ، کاغذ، قیچی هست که با پایتون نوشته شده و رابط گرافیکی‌ش رو با Tkinter ساختیم.
کاربر یکی از گزینه‌ها رو انتخاب می‌کنه و کامپیوتر هم به صورت تصادفی یه انتخاب انجام می‌ده، بعد نتیجه‌ی بازی و امتیازها نمایش داده می‌شه.
________________________________________
2.	ابزارهایی که استفاده کردیم
•	Python
•	Tkinter (برای ساخت رابط گرافیکی)
•	random (برای انتخاب تصادفی حرکت کامپیوتر)
________________________________________
3.	نحوه کار بازی
کاربر با دکمه‌ها یکی از گزینه‌های زیر رو انتخاب می‌کنه:
•	سنگ
•	کاغذ
•	قیچی
کامپیوتر هم یه انتخاب تصادفی انجام می‌ده و بعد طبق قوانین بازی نتیجه مشخص می‌شه.
قوانین:
•	سنگ، قیچی رو می‌بره
•	قیچی، کاغذ رو می‌بره
•	کاغذ، سنگ رو می‌بره
•	اگر هر دو یکی باشن -> مساوی
________________________________________
4.	امکانات پروژه
•	انتخاب حرکت با دکمه
•	نمایش انتخاب کاربر و کامپیوتر
•	نمایش نتیجه (برد / باخت / مساوی)
•	سیستم امتیازدهی (امتیاز کاربر و کامپیوتر)
•	دکمه‌ی ریست برای شروع دوباره
•	دکمه خروج از برنامه
________________________________________
5.	بخش‌های اصلی کد
کد ما چند بخش مهم داره:
6.	لیست انتخاب‌ها
یه لیست داریم که گزینه‌های ممکن رو نگه می‌داره و کامپیوتر ازش انتخاب می‌کنه:
choices = ["سنگ", "کاغذ", "قیچی"]
7.	تابع play(user_choice)
این تابع وقتی اجرا می‌شه که کاربر یکی از دکمه‌ها رو بزنه.
توی این تابع:
•	انتخاب کامپیوتر با random.choice انجام می‌شه
•	نتیجه‌ی بازی بررسی می‌شه
•	امتیازها تغییر می‌کنن
•	همه چیز روی صفحه آپدیت می‌شه
3.	تابع update_score(user_win)
این تابع فقط برای امتیازدهی استفاده می‌شه.
اگر کاربر برده باشه امتیاز خودش زیاد می‌شه و اگر نه، امتیاز کامپیوتر.
4.	تابع reset_game()
این تابع بازی رو کامل به حالت اول برمی‌گردونه:
•	امتیازها صفر می‌شن
•	متن‌ها ریست می‌شن
•	دوباره بازی آماده شروعه
________________________________________
6.	رابط گرافیکی
پنجره‌ی اصلی بازی شامل این بخش‌هاست:
•	عنوان بازی
•	دکمه‌های انتخاب (سنگ / کاغذ / قیچی)
•	نمایش انتخاب کاربر و کامپیوتر
•	نمایش نتیجه
•	نمایش امتیازها
•	دکمه reset
•	دکمه exit
________________________________________
7.	اجرای پروژه
برای اجرا فقط کافیه فایل رو با پایتون اجرا کنید:
python rps_game.py
